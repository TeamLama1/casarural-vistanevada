# Casa Rural Vista Nevada

Un sitio web moderno y responsivo para Casa Rural Vista Nevada, un alojamiento rural de calidad en un entorno natural privilegiado.

## 🏔️ Descripción

Casa Rural Vista Nevada es un refugio de paz ubicado en plena naturaleza, rodeado de montañas y paisajes espectaculares. Nuestro sitio web presenta el alojamiento rural, las instalaciones, actividades disponibles y proporciona una forma fácil de contactar y solicitar información sobre reservas.

## ✨ Características

- **Diseño Responsivo**: Adaptado para dispositivos móviles, tablets y ordenadores
- **Navegación Intuitiva**: Menú de navegación fijo con smooth scrolling
- **Formulario de Contacto**: Sistema de contacto con validación
- **Galería Interactiva**: Galería de imágenes con efecto lightbox
- **Secciones Completas**:
  - Hero section con call-to-action
  - Información sobre el alojamiento
  - Galería de imágenes
  - Actividades y entorno
  - Formulario de contacto
  - Footer informativo

## 🚀 Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos modernos con Flexbox y Grid
- **JavaScript**: Funcionalidades interactivas
- **Google Fonts**: Tipografías Playfair Display y Open Sans

## 🌐 Uso

1. Abrir `index.html` en cualquier navegador web moderno
2. O servir los archivos desde un servidor web local:
   ```bash
   python3 -m http.server 8000
   ```
   Luego visitar `http://localhost:8000`
